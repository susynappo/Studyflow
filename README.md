\# StudyFlow



StudyFlow is a smart study plan platform designed to help students organize and track their study progress in an intuitive and interactive way.



\## Features



\- \*\*Add Exams:\*\* Users can add exams with details such as CFU, difficulty, and exam date.

\- \*\*Track Progress:\*\* Each exam has a dynamic progress bar showing completion percentage.

\- \*\*Weekly Plan:\*\* Automatically generates a weekly study plan based on remaining hours and exam difficulty.

\- \*\*Statistics:\*\* Visual charts show progress across all exams.

\- \*\*Motivational Interface:\*\* Animated dashboard and emojis to make studying more engaging.
